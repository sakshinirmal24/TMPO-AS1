# LNC2CANCER 3.0 Database Information

• Full Database Name: LNC2CANCER 3.0  
• Version Used: Latest web version (2025)  
• Access Date(s): 24-11-2025  
• Persistent Identifier/URL: http://www.bio-bigdata.net/lnc2cancer/  
• Description of Data Used: Used to explore experimentally supported associations between long non-coding RNAs (lncRNAs) and various cancers, with a specific focus on LUAD-related lncRNAs and their functional roles.  
• Key Parameters/Filters Applied: LUAD-specific lncRNAs, experimentally validated associations prioritized, literature-supported interactions.
