# ENCORI Database Information

• Full Database Name: ENCORI (The Encyclopedia of RNA Interactomes)  
• Version Used: Latest web version (2025)  
• Access Date(s): 18-11-2025  
• Persistent Identifier/URL: https://starbase.sysu.edu.cn/  
• Description of Data Used: Used to explore RNA–RNA and RNA–protein interactions, including miRNA–mRNA and lncRNA–miRNA networks in LUAD.  
• Key Parameters/Filters Applied: LUAD dataset, interaction confidence score filtering, experimentally supported interactions.
