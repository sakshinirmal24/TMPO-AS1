# MIRNET Database Information

• Full Database Name: MIRNET (miRNA Interaction Network Database)  
• Version Used: Latest web version (2025)  
• Access Date(s): 20-11-2025  
• Persistent Identifier/URL: https://www.mirnet.ca/  
• Description of Data Used: Used to explore miRNA–target gene interaction networks, including experimentally validated and predicted miRNA–mRNA relationships relevant to LUAD.  
• Key Parameters/Filters Applied: LUAD-related genes, high-confidence interaction filtering, experimentally supported interactions prioritized.
